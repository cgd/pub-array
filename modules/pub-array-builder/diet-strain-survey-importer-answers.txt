Diet Strain Survey Database
        This web application provides access to microarray gene expression\
        results for the Strain Survey Microarray Experiment detailed in\
        Shockley et al "Using Hepatic Gene Expression and Phenotypes to Explore\
        Atherogenic Diet in a Mouse Strain Survey", which has been submitted for\
        publication. For more information on how to use this application and\
        interpret the results please see the\
        <a href="StrainSurveyHelpFile.pdf">help documentation</a> [PDF].
